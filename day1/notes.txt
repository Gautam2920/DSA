1. What is a Data Structure?
-Way of organising the data in a computer for access and use.
-Types:
 -Linear Data Structure : Data arranged sequentially , elements attached.
 -Non Linear Data Structure : Not arranged sequentially.

2. Arrays?
-Linear Data Structure.
-Stores in contiguous memory 
 -C/C++: contiguous block of actual elements
 -Python / JS:
  -The array/list stores references contiguously
  -The actual objects may live elsewhere
-Offers:
 -Random access
 -Cache Freindliness
-Why O(1)?
 -Calculation > Searching
  -Formula : Address(arr[i]) = Base_Address + ( i Ã— Size_of_Element )
-Limitation?
 -Fixed Size
 -Contiguous Memory location
 -Costly insertion and deletion
 -Poor flexibility
-Dynamic vs Static Array?
Aspect	        Static Array	Dynamic Array
Memory Location	Stack	        Heap
Size Decision	Compile-time	Runtime
Resizing	    Not possible	Possible (costly)
Deallocation	Automatic	    Manual
Speed	        Faster	        Slightly slower
Risk	        Stack overflow	Memory leak
